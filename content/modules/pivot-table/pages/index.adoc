数据透视表（PivotTable）扩展组件为 Jmix UI 提供了一个具有拖拽功能的 `PivotTable` 组件，可以将数据集转换为汇总表，并使用 UI 中的二维拖拽进行操作。

该扩展组件是基于 https://github.com/nicolaskruchten/pivottable[PivotTable.js^] JavaScript 库。更多示例可以在作者的 http://nicolas.kruchten.com/pivottable/examples/[网站^] 查看。

image::overview-embedded-pivottable.png[align="center",width="1063"]

数据透视表扩展组件的关键功能包括：

* *交互式透视表组件*：提供一种交互式的可视化方式来浏览和分析数据。

* *拖拽功能*：通过直观的拖拽轻松操作表格的结构，或进行分组。

* *数据聚合*：通过计算总和、均值、计数等聚合运算进行数据探索。

* *基于 PivotTable.js*：基于强大的外部库构建，实现高级功能和更大的灵活性。
