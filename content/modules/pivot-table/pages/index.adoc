Дополнение PivotTable предоставляет компонент `PivotTable` для Jmix UI с функциональностью перетаскивания, который позволяет превратить набор данных в сводную таблицу и манипулировать им с помощью интуитивного интерфейса.

Дополнение основано на внешней библиотеке JavaScript https://github.com/nicolaskruchten/pivottable[PivotTable.js^]. Вы можете найти больше примеров PivotTable на http://nicolas.kruchten.com/pivottable/examples/[веб-сайте^] его автора.

image::overview-embedded-pivottable.png[align="center",width="1063"]

Ключевые особенности дополнения PivotTable:

* *Интерактивный компонент PivotTable*: Обеспечивает визуальный и интерактивный способ исследования и анализа данных.

* *Функциональность перетаскивания*: Легко манипулировать структурой таблицы и группировкой с помощью интуитивных действий перетаскивания.

* *Агрегирование данных*: Вычислять суммы, средние значения, количество и другие агрегаты, чтобы получить ценную информацию из ваших данных.

* *Использование PivotTable.js*: Основано на мощной внешней библиотеке для расширенных функций и гибкости.
