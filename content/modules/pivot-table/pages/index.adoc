The PivotTable add-on provides a `PivotTable` table component for Jmix UI with drag-and-drop functionality that enables turning a data set into a summary table and manipulating it using 2D drag-and-drop UI.

The add-on is based on the external https://github.com/nicolaskruchten/pivottable[PivotTable.js^] JavaScript library. You can find more examples of PivotTable on its authorâ€™s http://nicolas.kruchten.com/pivottable/examples/[website^].
