= 日历

日历扩展组件提供对 JavaScript https://fullcalendar.io/[FullCalendar] 的无缝集成，这是一个构建交互式日历的流行库。

扩展组件包括一个 `FullCalendar` 组件，支持以不同的显示模式（日、周、月等）在日历中显示事项。然后，用户可以通过以下方式与这些事项互动：

* *拖拽*：在日历内更改事项的日期或时间。
* *调整大小*：通过拖拽可视化元素的边缘更改事项的时长。
