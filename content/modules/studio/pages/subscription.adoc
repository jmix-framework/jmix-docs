= 账号和订阅

使用 Jmix Studio 时，需要登录您的 Jmix 账号。创建账号只需提供可用的 email 地址。

此外，您还可以购买订阅，以解锁 Studio 的高级功能并可使用商业扩展组件。在具有最多 10 个实体和角色的小型项目中，无需订阅即可使用高级 Studio 功能。

[[sign-in-to-account]]
== 登录账号

当您第一次运行 Jmix Studio 时，会展示 *Jmix Sign In* 对话框。在这个对话框中，可以登录账号或者注册一个新的账号。

image::sign-in-dialog.png[align="center",width="496"]

点击 *Sign In / Register* 按钮。将会切换至系统浏览器登录：

image::ROOT:store/login-form.png[align="center",width="1549"]

* 如果您没有 Jmix 账号，点击 *Register* 链接。在注册表单中，填写 *First name*、*Last name*、*Email* 和 *Password* 字段，接受隐私协议，然后点击 *Register* 按钮。
+
image::ROOT:store/registration-form.png[align="center",width="1549"]
+
NOTE: 您输入的电子邮件地址将作为登录用户名以及您账号的主要邮件地址。
+
当提交了注册表单后，需要通过邮件激活您的账号。
// +
// image::email-verification.png[align="center",width="936"]
// +
在邮箱中找到验证邮件并点击邮件中的链接完成注册。
// +
// Go back to the IDE and click the *Log in* button in the *Email Verification* window.

* 如果您已经有了 Jmix 账号，输入 *Email* 和 *Password* 并点击 *Sign In* 按钮。

[[subscription]]
== 订阅

Jmix 提供两种类型的商业订阅计划：*快速开发版（RAD）* 和 *企业开发版（Enterprise）*。免费计划和商业计划的比较可以参考 Jmix 网站： https://www.jmix.cn/subscription-plans-and-prices/[订阅计划和价格^]。

[[rad-subscription]]
=== 快速开发版（RAD）

快速开发版（Rapid Application Development）订阅可以使用下列 Studio 高级功能：

* xref:studio:entity-designer.adoc[实体和枚举设计器]
* xref:studio:data-stores.adoc#data-store-actions[自动生成 Liquibase 变更日志]
* xref:studio:view-designer.adoc[视图设计器]
* xref:studio:menu-designer.adoc[主菜单设计器]
* xref:studio:role-designer.adoc[角色设计器]
* xref:studio:fetch-plan-designer.adoc[Fetch plan 设计器]
* JPQL 查询语句设计器

NOTE: 上面列出的所有高级功能都可以在不超过 10 个实体和角色的小项目中免费使用。

当第一次登录 Studio 时，即获得快速开发版的试用订阅。试用订阅的有效期是 28 天，可以在任何大小的项目中使用 Studio 的高级功能。

image::jmix-subscription.png[align="center",width="612"]

试用结束后，Studio 只能在小项目中免费使用高级功能。

[[enterprise-subscription]]
=== 企业开发版（Enterprise）

企业开发版除了包含上述的快速开发版的所有功能之外，还可以在 Studio 中使用商业扩展组件，包括：xref:bpm:index.adoc[]、xref:business-calendar:index.adoc[]、xref:maps:index.adoc[]、xref:notifications:index.adoc[]、xref:webdav:index.adoc[] 以及 xref:studio:bpmn-designer.adoc[]。

商业扩展组件存储在单独的制件仓库中，当从 xref:studio:marketplace.adoc[市场] 安装商业扩展组件时，Studio 会自动在 `build.gradle` 添加对应的仓库。此外，Studio 也会将登录仓库的凭证写入 `.gradle/gradle.properties` 文件。了解仓库和凭证的更多信息，请参考各个商业组件的安装说明。

可以按照 xref:ROOT:account-management.adoc#enterprise-trial[账号、账单和订阅管理] 的说明获取企业版试用许可。

[[subscription-information]]
=== 订阅信息

*Jmix Subscription* 可以查看账号和订阅的信息。该窗口通过 Studio 中 *Jmix* 工具窗口的 *Settings* (image:icons/gear.svg[]) -> *Account Information* 打开。

image::jmix-account-info.png[align="center", width="624"]

账号信息也显示在 xref:studio:welcome.adoc[Welcome] 界面：

image::account-info-welcome-screen.png[align="center", width="1028"]

参阅 xref:jmix:ROOT:account-management.adoc[账号、账单和订阅管理]。
