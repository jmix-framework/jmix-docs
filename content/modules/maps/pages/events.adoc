= Events

The map and features has the following events.

=== MapClickEvent

[[SourceFeatureSelectEvent]]
=== SourceFeatureSelectEvent
An event is fired when features are selected or deselected. Note that only the newly selected or deselected features will be included in the event. For instance, if one feature is currently selected out of a total of two, and the last one is selected, only the newly selected feature will be included in the event.
