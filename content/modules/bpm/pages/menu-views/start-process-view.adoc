= Start Process

This view allows users to start processes where they set xref:bpmn/bpmn-process.adoc#starter-candidates[candidates].
If the process has a starting form, it will be displayed before the process starts.

image::bpm-menu-views/start-process-view.png[,800]

To have access to this view, the user should be assigned to the role *BPM: process actor*.
