= Decision Tables

This administrative view contains a list of deployed decision tables and allows you to open them in modeler,
view details, and upload from the XML file.

By default, the list is empty. Click the *Apply* button to see all decision tables or use a filter.
To see all the decision table versions, uncheck the *Last versions only* checkbox.

image::bpm-menu-views/decision-tables.png[,800]

In the *Decision editor* view, you can inspect the decision table in the XML format.
Besides, you can delete deployed table.
