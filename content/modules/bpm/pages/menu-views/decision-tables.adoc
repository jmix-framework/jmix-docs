= 决策表

这个管理员视图包含部署的决策表，并支持通过建模器打开、
查看详情或上传 XML 格式的决策表文件。

列表默认为空。点击 *应用（Apply）* 按钮查看所有决策表或设置过滤器。
如需查看决策表的所有版本，反选 *仅显示最新版本（Last versions only）* 即可。

image::bpm-menu-views/decision-tables.png[,800]

在 *决策编辑器（Decision editor）* 视图中，可以查看 XML 格式的决策表。
并且可以删除已部署的决策表。
