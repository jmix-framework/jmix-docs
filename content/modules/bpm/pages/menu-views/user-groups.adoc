= 用户组

在 *用户组（User groups）* 界面，可以管理用户分组，
用户组可以作为 xref:bpm:bpmn/bpmn-user-task.adoc#candidates-by-user-groups[用户任务] 执行人的候选。
一个用户可以属于多个组。

image::bpm-menu-views/user-groups-list.png[,800]

用户组可以是下列类型之一：

* *Users*，该类型的分组，要求显式地选择一组用户。
* *Roles*，该类型的分组，要求定义一组安全角色。
如果用户有其中一个角色，则认为用户属于该组。
* *All users*，这个分组自动包含所有用户。
