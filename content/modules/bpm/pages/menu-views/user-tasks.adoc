= User Tasks

This view displays all active user tasks.
At the top of the screen, you can filter tasks using various criteria: process definition,
task assignee, candidate user, or candidate group.
By default, the list is empty. Click the *Apply* button to see all tasks or set up filters.


image::bpm-menu-views/user-tasks-list.png[,800]

You can open the process instance the task belongs to, or reassign a task performer.
