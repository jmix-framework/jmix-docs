= 实体选取操作
:page-aliases: actions/standard-actions/entity-clear-action.adoc, actions/standard-actions/entity-lookup-action.adoc, actions/standard-actions/entity-open-action.adoc, actions/standard-actions/entity-open-composition-action.adoc

实体选取操作可在 xref:vc/components/entityPicker.adoc[] 和 xref:vc/components/entityComboBox.adoc[] 组件使用。这些组件实现了 `EntityPickerComponent` 接口。

当为这些组件添加实体选取操作后，该操作展示为组件内部的一个按钮。

下面是标准的实体选取操作类型：

[[entity_clear]]
== entity_clear

操作类：`EntityClearAction`

清除组件值。

如果组件展示一对一的组合实体，则清除时，也会从 xref:data/data-context.adoc[] 的保存集合中删除（在实体详情视图中点击 *OK* 按钮时发生）。

[[entity_lookup]]
== entity_lookup

操作类：`EntityLookupAction`

在实体列表视图中选择一个实体实例并设置给组件。

[[entity_open]]
== entity_open

操作类：`EntityOpenAction`

打开当前组件中选择实体的详情视图。

[[entity_openComposition]]
== entity_openComposition

操作类：`EntityOpenCompositionAction`

打开组件中选择的一对一组合实体的详情视图。如果此时没有关联实体（组件值为空），则会在打开详情视图时创建一个新的实例，并通过详情视图保存。
