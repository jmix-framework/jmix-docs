= 值选择操作
:page-aliases: actions/standard-actions/value-clear-action.adoc

值选择操作支持对 xref:vc/components/valuePicker.adoc[] 和 xref:vc/components/multiValuePicker.adoc[] 的数据进行操作。这些组件实现了 `PickerComponent` 接口。

示例：

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/valuepicker/value-picker-view.xml[tags=basics]
----

image::visual-components/components/value-picker-basic.png[align="center",width="210"]

有两种标准的操作：

[[value_clear]]
== value_clear

操作类：`ValueClearAction`

清除组件的值。

[[multi_value_select]]
== multi_value_select

操作类：`MultiValueSelectAction`

对 `multiValuePicker` 组件，打开一个弹窗并将选择的值列表设置给组件。

请参阅 xref:vc/components/multiValuePicker.adoc#predefined-action[multiValuePicker] 了解详情。
