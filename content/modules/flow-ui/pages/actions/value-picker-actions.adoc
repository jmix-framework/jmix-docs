= Действия компонентов выбора значений
:page-aliases: actions/standard-actions/value-clear-action.adoc

Действия компонентов выбора значений работают с содержимым компонентов xref:vc/components/valuePicker.adoc[] и xref:vc/components/multiValuePicker.adoc[]. Эти компоненты реализуют интерфейс `PickerComponent`.

Например:

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/valuepicker/value-picker-view.xml[tags=basics]
----

image::visual-components/components/value-picker-basic.png[align="center",width="210"]

В данной категории есть два стандартных действия.

[[value_clear]]
== value_clear

Класс действия: `ValueClearAction`

Очищает значение компонента.

[[multi_value_select]]
== multi_value_select

Класс действия: `MultiValueSelectAction`

Для компонента `multiValuePicker`, открывает диалог и устанавливает список выбранных значений в компоненте.

Подробнее см. раздел xref:vc/components/multiValuePicker.adoc#predefined-action[multiValuePicker].
