`details` 是一个可展开的面板，用于展示或隐藏一些内容。

* XML 元素：`details`

* Java 类：`JmixDetails`

== 基本用法

使用 `details` 隐藏一些附加信息或可选的操作。`details` 可以包含任何组件或布局。
