`scroller` 是一个支持内容区域滚动的布局容器。

* XML 元素：`scroller`

* Java 类：`JmixScroller`

== 基本用法

`scroller` 支持在一个比较小的区域展示需要大空间的内容。其内部可滚动区域放置的组件必须由一个布局组件包装，例如，xref:flow-ui:vc/layouts/vbox.adoc[]。

[[scrollBarsDirection]]
=== scrollBarsDirection

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://vaadin.com/docs/latest/components/scroller/#scroll-direction" class="vaadin-docs-btn" target="_blank">Vaadin</a>
</div>
++++

定义当内容超出区域时可以使用哪些方向的滚动。支持：

* `VERTICAL` -- 启用垂直滚动。
* `HORIZONTAL` -- 启用水平滚动。
* `BOTH` -- 启用垂直和水平滚动。这是默认值。
* `NONE` -- 禁用滚动。
