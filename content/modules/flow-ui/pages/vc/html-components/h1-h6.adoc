= h1-h6

HTML 的 `h1` 至 `h6` 元素用于展示 6 个级别的标题。

* XML 元素：`h1`、`h2`、`h3`、`h4`、`h5`、`h6`

* Java类：`H1`、`H2`、`H3`、`H4`、`H5`、`H6`

== 基本用法

使用不同级别的标题组织内容的结构。高级别的标题标注重要的区域可以便于用户找到重要的内容。

----

[[themeNames]]
=== themeNames

为组件添加主题。支持：

* `normal`
* `success`
* `error`
* `contrast`
* `primary`
* `small`
* `pill`
