= nativeLabel 原生标题
:page-aliases: vc/html-components/label.adoc

`nativeLabel` 对应 HTML 的 `label` 元素，可用于为组件创建标题。

* XML 元素：`nativeLabel`

* Java 类：`NativeLabel`

== 基本用法

`nativeLabel` 组件可以替换 Jmix 中大多数 UI 组件的 xref:vc/common-attributes.adoc#label[label] 属性。

如需为某个组件设置标题，需要将 label 的 <<setFor, setFor>> 属性设置为目标组件的 id。例如，为 `checkbox` 设置标题：

[source, xml, indent=0]
----
include::example$onboarding/src/main/resources/com/company/onboarding/view/htmlcomponent/label/label-view.xml[tags=basics]
----

image::visual-components/html-components/label-basic.png[align="center", width="185"]

将 label 与组件关联可以使得辅助功能技术能正确地解析组件的标题，比如屏幕阅读器。

如需添加不与组件关联的文本，请使用 `span` 或 `div` 组件。

[[themeNames]]
== 样式版本

使用 xref:vc/common-attributes.adoc#themeNames[themeNames] 属性来自定义组件的颜色、大小和形状。首先使用 `themeNames = "badge"` 将组件转换为徽章。然后，使用其他主题来增强组件的视觉效果。

[[whiteSpace]]
=== whiteSpace

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/white-space" class="mdn-docs-btn" target="_blank">MDN</a>

定义组件的 `white-space` 样式值。支持：

* `NORMAL`
* `NOWRAP`
* `PRE`
* `PRE_WRAP`
* `PRE_LINE`
* `BREAK_SPACES`
* `INHERIT`
* `INITIAL`
