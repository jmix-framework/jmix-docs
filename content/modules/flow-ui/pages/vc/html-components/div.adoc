= div

`div` 是一个支持放入相关联组件的容器。如果没有配置属性，或内部无组件，则显示为空白。

* XML 元素：`div`

* Java 类：`Div`

[[themeNames]]
=== themeNames

为组件添加主题。支持：

* `badge`
* `normal`
* `success`
* `error`
* `contrast`
* `primary`
* `small`
* `pill`

只有将 `badge` 与其他主题组合才能生效，如 <<variants, 上面>> 所示。

[[whiteSpace]]
=== whiteSpace

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/white-space" class="mdn-docs-btn" target="_blank">MDN</a>

定义组件的 `white-space` 样式值。支持：

* `NORMAL`
* `NOWRAP`
* `PRE`
* `PRE_WRAP`
* `PRE_LINE`
* `BREAK_SPACES`
* `INHERIT`
* `INITIAL`
