= HTML 组件

HTML 组件支持在视图内使用标准的 HTML 元素。

TIP: 或者，使用 xref:vc/components/html.adoc[html] 组件可以添加任意标准 HTML 元素或 HTML 代码块。

|`div`

|`em`

|`footer`

|`h1` ... `h6`

|`header`
