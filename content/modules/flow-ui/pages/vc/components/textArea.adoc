= textArea 文本域

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://demo.jmix.io/ui-samples/sample/text-area-simple" class="live-demo-btn" target="_blank">在线示例</a>
</div>
++++

一个支持输入多行文本的区域。

* XML 元素：`textArea`

* Java 类：`JmixTextArea`

== 基本用法

当输入可能跨越多行（例如评论或描述）时，请使用 `textArea`。

image::visual-components/components/text-area-basic.png[align="center", width="200"]

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/textarea/text-area-view.xml[tags=basics]
----

如果未设置固定高度，`textArea` 会根据内容自动调整高度。默认高度和最小高度是两行文字。

== 数据绑定

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://demo.jmix.io/ui-samples/sample/text-area-dataaware" class="live-demo-btn" target="_blank">在线示例</a>
</div>
++++

数据绑定是指将可视化组件与数据容器相关联。这样可视化组件或数据容器的变更都会触发彼此的更新。参考 xref:data/data-examples.adoc[]。

数据感知的 `textArea` 示例：

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/textarea/text-area-view.xml[tags=data-binding]
----

确保需要绑定的实体属性为字符串类型。与 xref:vc/components/textField.adoc[] 不同，`textArea` 是专门为处理文本数据而设计的。

== 样式版本

=== 对齐

有三个选项：`align-left`（默认）、`align-right`、`align-center`。
