= multiSelectListBox 多选列表

`multiSelectListBox` 支持用户通过复选框在一个可滚动的列表中选择多个数据。

* XML 元素：`multiSelectListBox`

* Java 类：`JmixMultiSelectListBox`

== 基本用法

使用 xref:vc/components/entityComboBox.adoc#itemsContainer[itemsContainer] 或 xref:vc/components/comboBox.adoc#itemsEnum[itemsEnum] 属性创建选项列表：

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/multiselectlistbox/multi-select-list-box-view.xml[tags=data;hobbiesDc;layout;basics]
----

<1> `Hobby` 实体的 `CollectionContainer`。
<2> `multiSelectListBox` 使用 `hobbiesDc` 作为选项列表数据容器。
