提示窗是小的弹窗，用于在鼠标悬停或者键盘聚焦时为 UI 元素提供额外的信息。

XML 元素：`tooltip`

== 基本用法

`tooltip` 是基本上所有组件和布局的内部元素。

TIP: 如需通过 Jmix Studio 添加 XML 中的 `tooltip` 元素，请在界面 XML 中，或者 *Jmix UI* 层级结构中选择需要添加提示的 UI 元素，然后点击 *Jmix UI* 组件面板的 xref:studio:view-designer.adoc#component-inspector-add-button[Add]*->Tooltip* 进行添加。
