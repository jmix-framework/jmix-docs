= tooltip

Tooltips are small pop-ups that can provide additional information about a UI element on hover and keyboard focus.

XML element: `tooltip`

== Basics

`tooltip` is a nested element of almost all components and layouts.

TIP: To add a nested `tooltip` element in Jmix Studio, select the UI element in the screen descriptor XML or in the *Jmix UI* hierarchy panel and click on the xref:studio:screen-designer.adoc#component-inspector-add-button[Add]*->Tooltip* button in the *Jmix UI* inspector panel.
