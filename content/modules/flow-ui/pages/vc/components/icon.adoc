[[ClickEvent]]
=== ClickEvent

`com.vaadin.flow.component.ClickEvent` 当用户点击组件时发送。处理方法需要指定下列三种类型的点击之一。

* click - 点击时触发

* singleClick - 在一定的延时后触发，以确保不是双击。

* doubleClick - 当双击组件时触发。
