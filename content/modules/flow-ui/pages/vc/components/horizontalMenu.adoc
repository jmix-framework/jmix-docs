= horizontalMenu - 水平菜单
:page-aliases: vcl/components/app-menu.adoc

`horizontalMenu` 组件用于展示包含层级下拉子菜单的水平菜单栏。

xref:studio:view-wizard.adoc[视图创建向导] 中的 *Main view with top menu* 模板用的就是 `horizontalMenu`。如果需要用新的视图替换已有的主视图，需要在向导的第一步勾选 *Use as default main view*。之后，Studio 会替换所有视图 `@Route` 注解中的 `layout` 属性，并设置 xref:flow-ui:ui-properties.adoc#jmix.ui.main-view-id[jmix.ui.main-view-id] 应用程序属性为新视图的 id。

也可以与其他可视化组件一样，用在任何视图中。

* XML 元素：`horizontalMenu`

* Java 类：`HorizontalMenu`

[[loadMenuConfig]]
=== loadMenuConfig

`loadMenuConfig` 属性用来控制是否从 xref:flow-ui:menu-config.adoc[菜单配置] 中加载菜单结构。默认为 `true`，也就是说默认会从 `MenuConfig` 中加载菜单。如果需要使用不同的菜单项，需要先设置 `loadMenuConfig` 为 `false`。
