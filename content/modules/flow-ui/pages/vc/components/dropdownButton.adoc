[[dropdownIndicatorVisible]]
=== dropdownIndicatorVisible

设置下拉提示符的可见性。

默认值为 `true`。

[[openOnHover]]
=== openOnHover

如果 `openOnHover` 属性设置为 `true`，则当光标悬停或触屏点击时自动显示选项列表。

默认值为 `false`。
