= bigDecimalField 大数字段

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://demo.jmix.io/ui-samples/sample/big-decimal-field-simple" class="live-demo-btn" target="_blank">在线示例</a>
</div>
++++

`bigDecimalField` 与 xref:vc/components/textField.adoc[textField] 功能相似，但仅允许输入 `BigDecimal` 类型。

* XML 元素：`bigDecimalField`

* Java 类：`JmixBigDecimalField`

== 基本用法

`bigDecimalField` 的基本示例：

----

----

== 数据绑定

该组件支持绑定至 xref:data/data-containers.adoc[data container] 中的实体属性。

如需将 `bigDecimalField` 绑定至实体属性：

. 使用 xref:flow-ui:vc/common-attributes.adoc#dataContainer[dataContainer] 属性指定数据容器的名称。
. 使用 xref:flow-ui:vc/common-attributes.adoc#property[property] 属性指定实体属性的名称。

实体属性必须为 `BigDecimal` 类型。

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/bigdecimalfield/bigDecimalField-view.xml[tags=data;layout;data-binding]
----
