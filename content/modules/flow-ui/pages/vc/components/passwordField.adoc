= passwordField 密码输入

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://demo.jmix.io/ui-samples/sample/password-field-simple" class="live-demo-btn" target="_blank">在线示例</a>
</div>
++++

一个用于输入密码的组件。组件中输入的字符会默认进行掩码处理。用户可通过该组件设置或修改密码，或者在登录界面用于密码输入。

TIP: xref:vc/components/loginForm.adoc[loginForm] 组件包含一个密码输入控件。

* XML 元素：`passwordField`

* Java 类：`JmixPasswordField`

image::visual-components/components/password-field-basic.gif[align="center", width="225"]

`passwordField` 组件的属性与 xref:vc/components/textField.adoc[] 组件类似。但是，密码组件没有 `datatype` 属性，因为密码组件只能输入 `String`。
