== 基本用法

`radioButtonGroup` 的最基本用法是为实体属性选择一个枚举值。

image::visual-components/components/radio-button-group-basics.png[align="center", width="211"]

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/radiobuttongroup/radio-button-group-view.xml[tags=data;userDc;layout;basics]
----

上面的示例中，`User` 实体的 `onboardingStatus` 属性是一个 `OnboardingStatus` 枚举类型。

== XML 属性

在 Jmix 中，所有组件都有一些作用一致的 xref:vc/common-attributes.adoc[通用属性]。
下面是 `radioButtonGroup` 的特殊属性：

[cols="1h,5,1", grid=rows, frame=none]
|===
|*名称*
|*描述*
|*默认值*
|[[itemsContainer]]itemsContainer
|定义包含选项列表的数据容器。组件显示实体的实例名称。参阅 <<data-binding,数据绑定>>。
|
|[[itemsEnum]]itemsEnum
|定义用于创建选项列表的枚举类。参阅 <<items-enum,选项枚举>>。
|
|===
