Дополнение Kanban предоставляет компонент kanban-доски, визуально представляющий рабочий процесс проекта. Оно использует карточки для представления задач и колонки для представления различных этапов проекта, что позволяет пользователям быстро оценить прогресс. Дополнение основано на JavaScript-библиотеке https://htmlelements.com/demos/kanban/overview/[Smart Kanban^].

image::overview-embedded-kanban.png[align="center",width="890"]

Ключевые возможности дополнения Kanban:

* *Визуальная kanban-доска*: Полностью настраиваемый компонент xref:kanban:kanban-component.adoc[kanban]-доски, который можно интегрировать в любой экран Jmix.
* *Управление задачами*: Управление задачами на доске, включая назначение их пользователям, установку сроков и приоритетов.
* *Управление этапами*: Создание и управление этапами (xref:kanban:kanban-component.adoc#column[колонками]) на доске, представляющими различные фазы вашего рабочего процесса.
* *Функция перетаскивания*: Перемещение задач между этапами с легкостью, предоставляя удобный и интуитивно понятный интерфейс.
* *Опции настройки*: Настройка внешнего вида и поведения доски в соответствии с вашими потребностями и предпочтениями.
* *Интеграция с Jmix*: Дополнение Kanban легко интегрируется с фреймворком Jmix, позволяя вам использовать существующие данные и функциональность.

== Установка

[NOTE]
====
Это дополнение требует Enterprise https://www.jmix.io/subscription-plans-and-prices/[подписки^]. Если у вас нет подписки, см. раздел xref:ROOT:account-management.adoc#enterprise-trial[Пробная Enterprise подписка], чтобы узнать, как получить пробную версию.
====

Для автоматической установки через Jmix Marketplace следуйте инструкциям в разделе xref:ROOT:add-ons.adoc#installation[Дополнения].

Для ручной установки выполните следующие шаги.

. Настройте доступ к премиум-репозиторию.
+
include::ROOT:partial$premium-repo.adoc[]

. Добавьте зависимости в `build.gradle`:
+
[source,groovy,indent=0]
----
implementation 'io.jmix.kanban:jmix-kanban-flowui-starter'
----
