`CurrencyField` is a subtype of a xref:vcl/components/text-field.adoc[] designed for currency value input. `CurrencyField` is used by default in screens generated by Studio for the attributes annotated with `@CurrencyValue` annotation. It has a currency symbol inside the field and is aligned to the right by default.

image::vcl/components/currency-field.jpg[align="center"]

Component's XML-name: `currencyField`.

[[basics]]
== Basics

Basically, `CurrencyField` repeats the functionality of xref:vcl/components/text-field.adoc[]. You can manually set a xref:vcl/components/text-field.adoc#data-type[datatype] for the field, except that only the following numeric datatypes are supported:

* `decimal`
* `int`
* `long`
* `double`
* `float`

[[attributes]]
== Attributes

include::../xml-studio-tip.adoc[]

This component has three special attributes:

[[currency]]
* `currency` - defines text that will be the content of the currency label.

[[currency-label-position]]
* `currencyLabelPosition` - defines the position of the currency label inside the field. There are the following possible values:
** `RIGHT` - to the right from the text input component. The default value.
** `LEFT` - to the left from the text input component.

[[show-currency-label]]
* `showCurrencyLabel` - defines whether the currency label should be displayed.
